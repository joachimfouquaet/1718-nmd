Mezelf

Ik ben Joachim Fouquaet en studeer grafische en digitale media aan Artevelde hogeschool.

Mijn afstudeerrichting is crossmedia ontwerp. Deze richting heb ik gekozen omdat het in het verlengde ligt van wat ik in het middelbaar deed.

Mijn goede punten zijn dat ik goed met illustrator kan werken. In het middelbaar heb ik ook al een behoorlijke basis meegekregen i.v.m. programmatie. Zo heb ik geleerd met html, css en javascript te werken.
